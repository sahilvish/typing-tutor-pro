⌨️ Typing Tutor Pro

Vyapam Typing Tutor Pro ek modern, web-based application hai jo vishesh roop se CG Vyapam Data Entry Operator (DEO) aur Sahayak Grade 03 ke skill tests ki tayari ke liye banaya gaya hai. Ye macOS style UI ke sath aata hai aur typing muscle memory develop karne mein madad karta hai.

✨ Features

🎯 Specialized Layouts: Kruti Dev 010 (Legacy) aur Mangal Remington Gail (Unicode) ka full support.

🚀 Structured Lessons: Beginner se lekar Advanced level tak ke structured path (Home Row, Shift, Alt Codes).

☁️ Cloud Save: Firebase integration ke sath har user ka alag profile aur progress tracking.

🌓 Dark Mode: Raat mein bina aankhon par jor dale typing practice ke liye clean dark theme.

⌨️ Virtual Keyboard: Muscle memory build karne ke liye interactive QWERTY highlighting.

📊 Real-time Stats: Speed (WPM) aur Accuracy (%) ka live data.

🛠️ Tech Stack

Frontend: HTML5, CSS3 (Tailwind-like custom styles), JavaScript (Vanilla).

Backend: Firebase (Authentication & Firestore) for Cloud Progress saving.

Fonts: Custom @font-face integration for Kruti Dev compatibility.
